# Rebrand Step 3 — Wireframe-Level Page Structures (Canonical)

## Purpose
Define **exact page structures** (sections + order) before any design, copy polish, or Shopify implementation.

Rule: if a section does not reduce cognitive load, it does not exist.

---

## Page Map (Final)

1. Homepage
2. Home Styling — Landing
3. Home Styling — Tier Pages (3)
4. Gifting — Landing
5. איך זה עובד
6. אודות

Total: **6 page types** (intentionally small)

---

## 1. Homepage — Decision Split

### Section Order

1. **Hero**
   - One sentence
   - One still interior image
   - No movement

2. **Quiet Proof (3 lines max)**
   - פרחים שלא נעלמים
   - בלי תחזוקה
   - מסודר לשנה

3. **Two Paths**
   - Card A: לבית
     - Caption: עיצוב קבוע לבית
   - Card B: מתנות
     - Caption: תאריכים חשובים, מסודר מראש

4. **Soft Close**
   - Line: אפשר להפסיק לחשוב על זה

No footer content here beyond minimal nav.

---

## 2. Home Styling — Landing Page

### Section Order

1. **Hero**
   - Headline: עיצוב לשנה — בבית שלך
   - Sub: שלוש רמות. החלטה אחת.

2. **What This Is (Short)**
   - Paragraph (2–3 lines)
   - Explains presence, not flowers

3. **Tiers Overview (Cards, Not Comparison)**
   - Tier 1 card
   - Tier 2 card
   - Tier 3 card

Each card contains:
- One-line definition
- Descriptor (not features)
- ₪/day anchor

4. **Expectation Setting (Fairness Block)**
   - Text:
     אפשר לרענן את הסידור עד שלוש פעמים במהלך השנה — לפי הצורך.

5. **Soft CTA**
   - לבחור רמה

---

## 3. Home Styling — Tier PDP (Same Structure for All 3)

### Section Order

1. **Hero Image**
   - Interior context
   - No product isolation

2. **One-Line Definition**
   - Tier-specific, calm

3. **How It Feels Over a Year**
   - Timeline strip (Start → Mid → End)
   - No dates

4. **What’s Included (Plain)**
   - 3–4 bullets, non-numerical where possible

5. **Fairness Line (Mandatory)**
   - אפשר לרענן עד שלוש פעמים בשנה — כשמרגיש נכון.

6. **Price Framing**
   - ₪/day large
   - Annual total small

7. **Commitment Selector**
   - One option only
   - Annual

8. **Checkout**

---

## 4. Gifting — Landing Page

### Section Order

1. **Hero**
   - Headline: מסודר לשנה
   - Sub: שלושה תאריכים. החלטה אחת.

2. **Problem Statement (Short)**
   - Forgetting is the enemy

3. **Linear Flow (Visual Steps)**
   - קובעים → אנחנו זוכרים → זה מגיע

4. **Date Selector Block**
   - Three fields
   - Calm defaults

5. **Reassurance Line**
   - בלי תזכורות. בלי הזמנות ברגע האחרון.

6. **Price + Commit**
   - ₪/day
   - Annual

---

## 5. איך זה עובד

### Section Order

1. **Three Principles**
   - קובעים מראש
   - שקט לאורך השנה
   - ריענון כשצריך

2. **Home vs Gifting (Text Only)**
   - Two short paragraphs

3. **Fairness Paragraph (Explicit)**
   - Explains “up to three times” clearly

No diagrams. No FAQs explosion.

---

## 6. אודות

### Section Order

1. **What We Believe**
   - Calm authority

2. **Shiran (Human Anchor)**
   - One portrait
   - One paragraph

3. **Close**
   - Line: עיצוב שנשאר

---

## Step 3 Exit Check

You are done when:
- Every page has ≤8 sections
- No page explains itself twice
- You could explain the whole site in 30 seconds

This is now ready for Shopify section architecture.
