## 🧩 ELos Core Functions Across Tiers

This document outlines the foundational functions of ELos and how they evolve across Beginner, Intermediate, and Pro tiers. These functions define the core of the system's UX and determine the structural logic behind all menus and user interactions.

---

| Core Function   | Beginner Tier                            | Intermediate Tier                          | Pro Tier                                    |
|----------------|------------------------------------------|--------------------------------------------|---------------------------------------------|
| **Canvas**     | Create simple notes                      | Structured memory with tags and linking    | Advanced logic scaffolding and automation   |
| **Personas**   | Read-only, limited selection             | Selectable personas with context linking   | Full persona creation and chaining          |
| **Voice Modes**| Basic tone adjustments                   | Customizable voice profiles                | Dynamic voice modulation per context        |
| **Who Am I**   | Basic identity settings                  | Contextual role definitions                | Deep psychoanalytic profiling               |
| **Memory**     | View and submit preferences              | Context tagging and override logic         | Full memory editing and version control     |
| **Academy**    | Access to Phase 1 and 2 guides           | Intermediate-level courses and walkthroughs| Advanced system design and orchestration    |
| **Implementor**| Not available                            | Preview of tools                           | Full access to automation tools             |

---

### 🔁 Functional Naming Evolution
While the core functions remain consistent, the **menu labels may evolve** with tier progression to reflect the user’s level of engagement and capability:

| Core Function   | Beginner Label     | Intermediate Label    | Pro Label              |
|----------------|--------------------|------------------------|------------------------|
| Canvas         | Notes / Save       | Explorer / Memory Map  | System Logic Engine    |
| Personas       | Voice Filters      | Strategic Roles        | Behavioral Architects  |
| Voice Modes    | Tone Tweaks        | Voice Editor           | Multi-Context Voice AI |
| Who Am I       | About Me           | My Roles               | Identity Matrix        |
| Memory         | Preferences        | Memory Map             | Cognitive Control Hub  |
| Academy        | Learn              | ELos Academy           | Academy + Sim Lab      |
| Implementor    | (Hidden)           | Tool Previews          | Implementor Dashboard  |

---

### 🧭 Design Guideline
- Menus in Beginner are **simplified and instructional**
- Intermediate surfaces **structure and preview**
- Pro exposes **system-level control and automation**

This evolving taxonomy allows users to grow into the system, while each function retains conceptual continuity.
