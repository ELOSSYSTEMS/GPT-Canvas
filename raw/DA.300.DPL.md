# DABROO COURSE DEPLOYMENT PLAN

**Purpose**: Define a phased, structured rollout for the Dabroo Master Course across multiple user tiers and interface formats. This is the operational roadmap for turning the course into a real user experience, from MVP to full-blown educational system.

---

## PHASE 1: INTERNAL STRUCTURE FINALIZATION

**Goals**:
- Finalize module hierarchy
- Lock tone, voice, and instructional rhythm
- Ensure alignment with Dabroo product roadmap

**Tasks**:
- Review all modules in DA.100.CRS
- Add visuals, metaphors, and real prompt examples per module
- Embed active Canvas and Persona references

---

## PHASE 2: DELIVERY FORMATS

**1. Notion Wiki (Internal + Partner Beta)**
- Course stored as full structured workspace
- Embedded canvases as linked databases
- Tiered permission levels: public preview / Pro / Allon-only

**2. Dabroo In-App Wizard (Future State)**
- Modular onboarding in the Dabroo UI
- Persona-based guidance per module
- Contextual triggers (e.g., using prompts activates the Prompt Surgeon’s tips)

**3. Drip Email Series (Freemium/Pro)**
- Weekly breakdown of modules
- Each email = one use case with example + upsell CTA
- Links directly to Canvas-based resource center

**4. Premium Video Course** *(Phase 3)*
- 5–10 videos hosted on a locked domain
- Voiceover, screen sharing, prompt walkthroughs
- Bundle with Pro subscription or sell standalone

---

## PHASE 3: TECHNICAL INFRASTRUCTURE

**Key Components**:
- Trigger phrase loading ("Ready to Start", "Let’s Talk About Dabroo")
- Persona auto-summon during course usage
- Smart UI overlays for tooltips and embedded prompts
- Memory simulation scaffolding to resume from last point

**Tools**:
- Make (automation)
- Google Sheets (progress tracking)
- LocalStorage (in-browser resume logic)

---

## PHASE 4: TIERING + MONETIZATION

**Free Tier**:
- First 3 modules
- Email-only drip access
- No Whisper Mode, no persistent memory

**Pro Tier**:
- Full canvas + Notion access
- Advanced personas
- Whisper Mode, Recall Mode, Behavior Control Panel

**Consulting Tier**:
- Personalized interface buildout
- Custom personas for their brand
- Prompt surgery with Allon

---

## PHASE 5: FEEDBACK + ITERATION

**Internal QA**:
- Alpha user test (power user cohort)
- Prompt accuracy, tone consistency, usability

**User Feedback Channels**:
- Embedded feedback links in each module
- Slack/Telegram channel for real-time issue reports
- Feature request form linked to Notion board
