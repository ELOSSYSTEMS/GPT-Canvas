## PHASE 1 – SYSTEM GENESIS ARCHITECTURE

### 1.1 Incentive Scaffold Mapping
- Core question: *What behavior is this system paying people to repeat?*

  - Punishment structures
  - Edge-case exploit zones

### 1.2 Moral Axis Grid
- Determines ethical operating center:
  - Consequentialist?
  - Collectivist?
  - Libertarian?
  - Theocratic?

### 1.3 Myth Backbone Construction
- All civilizations run on myth:

  - Justification myth (why this power structure is “natural”)
  - External enemy (why it must survive)
