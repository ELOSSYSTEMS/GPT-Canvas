# Rebrand Step 2 — Current Site Audit (Delete / Keep / Transform)

## Purpose
This step removes *everything* that fights the new promise:
> “This is handled. Calmly.”

If something increases choice, urgency, comparison, or noise — it goes.

---

## 1. Global Deletions (Hard Cuts)

### Navigation
DELETE:
- Collections (as a concept)
- Bestsellers
- New arrivals
- Any promo-based links

KEEP:
- Home
- לבית (Home Styling)
- מתנות (Gifting)
- איך זה עובד
- אודות

---

### Homepage Sections
DELETE:
- Product grids
- Carousels
- Featured products
- Testimonials sliders
- USPs icon rows

TRANSFORM:
- Hero → single sentence + single image
- Any “why us” → 3 quiet statements max

---

## 2. Product Pages (PDPs)

DELETE:
- Individual bouquet PDPs
- Variant-heavy selectors
- Color swatches
- "Add to cart" urgency

REPLACE WITH:
- Tier pages (decision pages, not products)

One PDP per tier only.

---

## 3. Language & Copy

DELETE ALL INSTANCES OF:
- "מבצע"
- "חיסכון"
- "מנוי"
- "כל X חודשים"
- "קבלו"

REPLACE WITH:
- "מסודר"
- "אפשרות"
- "לאורך השנה"
- "כשמרגיש נכון"

---

## 4. Visual Noise

DELETE:
- Lifestyle shots with people
- Gift-wrapping imagery
- Seasonal graphics
- Badges, labels, stickers

KEEP:
- Still interiors
- One arrangement per frame
- Neutral daylight

---

## 5. Trust Elements (Reorder)

MOVE:
- Policies → footer only
- FAQs → calm explainer page

ADD:
- One paragraph explaining fairness:
  "אפשר לרענן עד שלוש פעמים בשנה — לפי הצורך, לא לפי לוח זמנים."

---

## 6. What Survives the Rebrand

KEEP:
- Brand name
- Core aesthetic
- Shiran as human anchor (but quieter)

TRANSFORM:
- Flowers → system
- Products → decisions
- Purchase → relief

---

## Step 2 Exit Check

You are done when:
- There is nothing to browse
- There is nothing to compare
- There is nothing to rush

If the site feels almost empty — you are doing it right.
