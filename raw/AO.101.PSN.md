---

## 🔄 LIFECYCLE STAGES

| Stage | Description |
|-------|-------------|
| **Drafted** | Idea or intention noted, not implemented |
| **Scaffolded** | Logic partially written or outlined, inactive |
| **Active** | Logic functional, persona callable and responsive |
| **Locked** | Fireproofed, fingerprinted, cannot mutate without override |
| **Deprecated** | Archived, invalidated, replaced, or corrupted |
