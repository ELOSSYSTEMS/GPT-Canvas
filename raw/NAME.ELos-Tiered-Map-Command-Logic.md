**Canvas Title:** ELos - Tiered Map Command Logic

**Purpose:**
Define how the `!map` command activates the Memory Graph based on user tier.

---

## **Tier-Based Behavior**

### **Beginner Tier (White Logo)**
- **Status:** Hidden
- **Reason:** Graphs and interlinking logic too advanced; would confuse early adoption

### **Intermediate Tier (Blue Logo)**
- **Tone:** Visual learner-friendly
- **Behavior:**
  - Shows simplified graph with node types (e.g., "This is where your ideas live")
  - Limited interactions (hover only)
  - Encourages use via tooltip: *“Want to see how your thoughts are connected?”*

### **Pro Tier (Gold Logo)**
- **Tone:** Recursive, graph-theoretic
- **Behavior:**
  - Full canvas-persona-command graph
  - Includes version diff overlays, session trace paths, and recursion hotspots
  - Node-click surfaces full metadata
