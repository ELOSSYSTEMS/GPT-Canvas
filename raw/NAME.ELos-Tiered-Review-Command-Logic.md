**Canvas Title:** ELos - Tiered Review Command Logic

**Purpose:**
Standardize the behavior of the `!review` command across all ELos tiers to support progressive familiarity with memory recall, decision checkpoints, and system outputs.

---

## **Tier-Based Behaviors**

### **Beginner Tier (White Logo)**
- **Tone:** Encouraging, simplified
- **Behavior:**
  - Shows latest 1–2 logged thoughts
  - Explains in plain language: *“Here’s what you’ve been thinking about lately.”*
  - Option to ask: *“Want to set a reminder or revisit this later?”*

### **Intermediate Tier (Blue Logo)**
- **Tone:** Clarity-focused
- **Behavior:**
  - Displays log summaries, tagged entries, and dates
  - Suggests grouping, deleting, or journaling follow-up
  - Mentions canvas titles softly (e.g., “Saved in: Daily Flow”)

### **Pro Tier (Gold Logo)**
- **Tone:** Audit-style, modular
- **Behavior:**
  - Surfaces multi-canvas log history by tag, timestamp, or persona link
  - Allows filtering by category, date, or trigger
  - Outputs in structured format (markdown, JSON)
