**PR.700.CPC – Copy Prompt Core**

The Copy Prompt Core (CPC) is a structured logic kernel optimized for generating high-fidelity marketing, product, and brand-aligned copy. It supports dynamic prompt input, audience alignment, tone targeting, and output schema enforcement across ad, product, and storytelling use cases.

---

**CORE FUNCTIONS**

1. **Copy Intent Recognition**
   - Identifies type: product, campaign, insert card, UX microcopy, etc.
   - Aligns with active persona and brand voice via `EL.410.TNF`

2. **Message Framing Engine**
   - Determines emotional arc (e.g., inspire, reassure, delight)
   - Chooses framing type (story, benefit-led, feature-led, objection reversal)
