**Canvas Title:** ELos - Intermediate Tier System Overview

**Purpose:**
Provide an instructional, no-nonsense introduction to ELos for users in the Intermediate Tier. Assumes basic comfort with ChatGPT, introduces more control and options without overwhelming.

---

## **What Is ELos?**

ELos is your **assistant control system**. It helps you:
- Customize how ChatGPT speaks and behaves
- Organize memory through modules (called canvases)
- Use commands to manage tasks, tone, and system state

You're no longer just chatting. You're starting to **guide the system**.

---

## **Why It Matters Now**

At this tier, you’re ready for more than casual use. ELos gives you:
- Deeper control (modes, memory, journaling)
- Smarter workflows (`!log`, `!whoami`, `!diff`)
- Context awareness and structured guidance

---

## **What’s Unlocked at Intermediate**
- Full command menu (except for hidden diagnostic tools)
- Canvas system: organize knowledge, ideas, or tasks
- Persona selector: change how your assistant thinks

**Try This:**
- Type `!mode` to switch behavior style
- Use `!diff` to compare canvas versions (Beta)
- Start a log with `!log` — all saved behind the scenes

---

## **What’s Still Hidden**

Advanced tools like:
- Shadow Mode (`!shadowlog`)
- AI Auditing Toolkit (`!audit`)
- Recursive Graph Mapping (`!map`)

These unlock only at Pro tier to avoid confusion.

---

You’re in the **Blue Tier**.  
You’ve got the tools. The system will now respond to more advanced commands and instructions.

## **Summary**
> ELos gives you more power, more clarity, and more control — without complexity. You're no longer a user. You're a builder.
