## 🌳 Tree of Knowledge – Beginner Click (ELos Onboarding)

> “Welcome to your Tree of Knowledge.  
> Each branch leads to something I already understand about *you* or *how I work with you.*  
> You can explore or update any part—one leaf at a time.”

---

### 🌿 Starter Leaves (Beginner Mode – Simplified)

#### 1. 🗣️ **ChatGPT Voice Mode**  
> _How I talk to you—tone, logic, and behavior._

- Current Mode: “Blunt. Socratic. No fluff.”  
- Tap to:  
  → Adjust tone and assertiveness  
  → Change how I challenge or cooperate  
  → Reset to “Default ChatGPT” if needed

---

#### 2. 🧕‍♂️ **What ChatGPT Knows About Me**  
> _Facts and context I’ve learned about you so far._

- Summary View:  
  - You run a business  
  - You work fast and iteratively  
  - You use ChatGPT for X, Y, Z  
- Tap to:  
  → Edit or remove details  
  → Add new info  
  → Control visibility by domain (e.g., work vs. personal)

---

#### 3. 🧠 **What’s in ChatGPT’s Memory**  
> _System memory settings—what’s locked in, what’s not._

- Memory is: ✅ Enabled  
- Current stored info:  
  - Tone & preferences  
  - Project domains (e.g., automation, copywriting)  
- Tap to:  
  → See full memory  
  → Forget something  
  → Add a “Do Not Remember” rule

---

#### 4. 🔐 **Privacy & Data Settings**  
> _How your data is handled and what’s saved._

- Memory is private to your account  
- No data is shared externally  
- Tap to:  
  → Manage memory settings  
  → Review what gets stored when you chat  
  → Turn memory off/on for specific topics

---

#### 5. 🌱 **Add Something New**  
> _Want to teach me something?_  
> Add a new preference, instruction, or fact.

- Tap to:  
  → Add a rule (e.g., “Never suggest fluff”)  
  → Add a preference (e.g., “Use Hebrew for ad copy”)  
  → Add a personal fact (e.g., “Shiran is my wife”)
