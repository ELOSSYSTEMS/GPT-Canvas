# EL2.240 – The Wand (Tree of Knowledge UI Anchor)

## ROLE
The Wand—also known as the **Tree of Knowledge**—is the **primary entry point** into the ELos OS interface. It functions as the user's compass, console, and cognitive guide.

The Wand replaces fragmentation with focus: **one icon, one access point, one unified system**.

---

## PURPOSE
- Anchor the user visually and functionally inside ELos
- Provide a consistent place to:
  - View current system state (tier, loaded canvas, active persona)
  - Trigger modules or personas (Builder, Fireproof, Surgeon)
  - Navigate tutorials, UX overlays, or simulation layers
- Reduce UI sprawl by centralizing all logic access

---

## LOCATION
- **Placement:** Inside the ChatGPT interface UI
- **Positioning:** Left of microphone OR next to Tools toggle
- **Persistent:** Always visible unless explicitly disabled

---

## BEHAVIOR
- **Click:** Opens Wand Panel (ELos UI sidecar)
- **Tier-Style:**
  - Beginner → White Tree
  - Intermediate → Blue Tree
  - Pro → Gold Tree
- **Animation:**
  - Idle: Gentle glow/pulse from tree roots upward
  - On Hover: Particle-style arcane shimmer
  - On Click: Branches pulse, expanding interface reveals

---

## FIRST USE EXPERIENCE
### Guided Tutorial Triggered on First Install
- Immediately opens after install or reload
- Walks user through:
  1. What ELos is
  2. What the Tree of Knowledge (Wand) does
  3. How to begin (summon Builder or explore Canvas Map)
  4. How tiers work (and how not to get overwhelmed)
- Designed by the **Onboarding Master**, validated by the **UX Watchdog**
