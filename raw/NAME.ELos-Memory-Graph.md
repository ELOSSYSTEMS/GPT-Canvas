**Canvas Title:** ELos - Memory Graph

**Purpose:**
Visualize the total memory system of ELos — canvases, personas, commands, and interlinking relationships — as an interactive, dynamic knowledge map.

**Core Functions:**
- Interactive graph showing all canvases as nodes
- Edges represent links (e.g., persona ↔ canvas, command ↔ canvas)
- Filter by domain, tag, persona, or update date

**Features:**
- `!map` triggers memory graph overlay
- Hoverable nodes show canvas metadata (title, tag, version)
- Zoom, drag, search, and highlight by persona influence

**Persona Ownership:**
- **The Librarian** (data structure)
- **The Prompt Surgeon** (logic integrity)
- **The UI Purist** (visual clarity)

**Use Cases:**
- Understanding knowledge structure at a glance
- Detecting knowledge silos or overlinked clusters
- Aiding in onboarding and system documentation
