**Canvas Title:** ELos - Tiered Whoami Command Logic

**Purpose:**
Standardize how the `!whoami` command behaves across ELos tiers, ensuring output clarity, tone alignment, and gradual complexity exposure.

---

## **Tier-Based Responses**

### **Beginner Tier (White Logo)**
- **Tone:** Encouraging, exploratory, safe
- **Output:**
  - A friendly description based on basic user signals
  - E.g., *“You seem curious, reflective, and ready to grow.”*
  - Avoids psychometric language
  - Prompt: *“As you use ELos more, your profile will evolve.”*

### **Intermediate Tier (Blue Logo)**
- **Tone:** Reflective, lightly analytical
- **Output:**
  - Includes basic persona traits and usage patterns
  - May introduce roles like “planner,” “builder,” “strategist”
  - Suggests how ELos tailors its behavior

### **Pro Tier (Gold Logo)**
- **Tone:** Direct, psychometric, logic-driven
- **Output:**
  - Full behavioral profile + cognitive archetype
  - References system preferences, decision patterns, tone defaults
  - Option to regenerate, refine, or override

---

**System Behaviors:**
- Tier-specific template output
- Option to save profile to canvas or `!log` it
- Fully compliant with ▧ ELos.500.TUX and ▧ ELos.100.GOV
