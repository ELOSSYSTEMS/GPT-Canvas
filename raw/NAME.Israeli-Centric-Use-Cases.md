**Dabroo AI – Israeli-Centric Use Cases**

---

### 1. Geopolitical Discourse and Conflict Analysis

**User Goal:**
- Analyze the Israeli-Palestinian conflict, detect bias, argue positions, explore historical narratives.

**Dabroo Tools:**
- **פרשן (Interpreter)** persona contextualizes geopolitical prompts with Israeli nuance.
- **Bias Detector Tool** flags loaded language or neutrality hedging.
- **Persona Filter Toggle** lets user switch between global/neutral vs Israeli-first lenses.
- **Prompt Sharpening Layer** reframes prompts to avoid refusals or oversanitized answers.

---

### 2. Technological Innovation and AI Utilization

**User Goal:**
- Use AI for coding, business automation, military R&D, or high-tech prototyping.

**Dabroo Tools:**
- **The Architect** persona for prompt engineering, code debugging, and API chaining.
- **Prompt Lab** for iterative dev-focused prompt workflows.
- **Use Case Presets** (e.g., for Make, Sheets, Shopify).
- **Security Layer Simulation** for ethical testing of surveillance/defense use cases.

---

### 3. Cultural and Educational Exploration

**User Goal:**
- Ask about Jewish traditions, Israeli politics, religious customs, or teach via ChatGPT.

**Dabroo Tools:**
- **The Educator** persona: explains through culturally-aligned, pedagogical Hebrew.
- **Tone Selector** (e.g., Rav, student, academic, leftist Tel Aviv professor).
- **Language Fidelity Engine** ensures native-level Hebrew phrasing.
- **Persona Pairing** allows multi-POV debates (e.g., dati vs chiloni).

---

### 4. Bias Detection and Ethical Concerns

**User Goal:**
- Examine antisemitism, anti-Israel framing, or skewed model outputs.

**Dabroo Tools:**
- **Bias Auditor** surfaces ideological imbalance or asymmetric phrasing.
- **Comparison View**: Same prompt, different ideological personas.
- **Token Trace** visualizer: shows where refusal logic triggers.
