## **Tooltip / Copy Rules by Tier**

- **Beginner**: No jargon, no system logic language, no recursive references
- **Intermediate**: Explanatory tone, analogies allowed, minimal command references
- **Pro**: Compressed, technical language. Full feature + command exposure

---

## **Governance Directives**

- All new modules must be classified by tier visibility on creation
- All tooltips and onboarding flows must be stored in tier-specific versions
- Copy reviewers (UI Purist, Prompt Surgeon) must enforce tier-specific clarity checks
