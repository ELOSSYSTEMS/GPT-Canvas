## ELos Academy – Beginner Guide to "What’s in ChatGPT’s Memory"

### 🧠 What Is Memory?
Memory is what separates a one-time assistant from a long-term ally.

When memory is on, ChatGPT can remember important facts and preferences you’ve shared. It can evolve. It can improve.

When memory is off, every session starts fresh—no history, no growth.

---

### 🔍 What’s Actually Stored?
Only high-signal, intentional data.

Examples of what memory might contain:
- "I prefer direct answers."
- "Use bullet points for summaries."
- "My company is focused on sustainability."

It doesn’t store:
- Everything you say
- Casual questions
- Sensitive data without your confirmation

---

### 🧾 What ChatGPT Allows
The memory system is simple but limited.

You can:
- Submit new preferences or facts using the memory panel
- Ask ChatGPT to forget something specific
- Turn memory off entirely if needed

But that’s where the limits begin. You don’t get full visibility. You can’t structure it. You can’t search it.

That’s where ELos changes everything.

> With ELos, you can create **canvases**—structured memory blocks that store tone, preferences, knowledge, or rules you define.

Canvases are:
- Visible
- Persistent
- Yours to shape

They don’t replace ChatGPT memory. They **surpass it**.
