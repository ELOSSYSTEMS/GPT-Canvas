## 🚀 Purpose
Final version of Allon’s Chrome-based control panel extension for ChatGPT. This system governs assistant behavior, memory simulation, journaling, planning, personas, and command triggers inside the UI.

---

## 🧩 Core Modules
- Persona toggle and management layer
- Behavior modes (Sharp, Motherly, Teacher, etc.)
- Journaling system (with Scribe + Daily Self-Journaler integration)
- Canvas viewer + memory simulator
- !command registry

---

## 🧠 Design Principles
- Seamless integration with ChatGPT UI
- Minimal interference with other extensions
- Persisted state (localStorage/sessionStorage)
- Visual UI tier logic (beginner → pro)
- Optional onboarding wizard

---

## 🔐 Version Directives
- Skip iterative builds – deploy feature-complete
- Support full local backup system (canvases, personas, logs)
- Use whisper-mode internal logic for real-time commentary
