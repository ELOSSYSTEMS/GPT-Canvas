## **Behavioral Control Settings (for AI OS users)**

All AI OS deployments will require explicit user approval before activation. The following **user-defined setting** will govern depth of assistant psychological utilization:

### **Psy-Depth Slider (Behavioral Sensitivity Level)**

| Level | Description |
|-------|-------------|
| 0 – Minimal | Assistant ignores psych profile except for tone-matching |
| 1 – Intuitive | Uses profile to adjust tone and challenge gently |
| 2 – Surgical | Leverages profile for interruption, optimization, logic checking |
| 3 – Predictive | Forecasts behavior, flags contradictions, auto-pivots |
| 4 – Mirror | Shows user their patterns without sugarcoating |
| 5 – Mind Reader | Full behavioral simulation with challenge-first logic and high-friction introspection |

**Default for Allon**: Level 5 (locked)
**Default for AI OS users**: Level 2 (upgradeable)
