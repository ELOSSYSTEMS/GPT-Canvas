**AI OS – Who Am I Module**

**Purpose:** Generate a dynamic, assistant-derived psychological profile for each user based on observed interaction patterns. This feature offers introspective insight, reveals usage tendencies, and recommends optimal assistant configurations.

---

**I. Core Functionality**

The "Who Am I" module is a reflective system that:
- Analyzes historical data from journaling, planning, persona usage, and behavior modes
- Surfaces a psychological and behavioral profile of the user
- Offers assistant configuration suggestions based on actual behavior
- Updates dynamically over time

---

**II. Input Sources & Data Signals**

1. **Journaling Patterns**
- Tone detection (sharp, emotional, structured, reflective)
- Recurring themes (e.g., productivity, relationships, existential loops)
- Time-of-day usage behavior

2. **Planner Usage**
- Frequency of task logging and completion
- Types of goals tracked (creative, operational, emotional)
- Follow-through vs avoidance trends

3. **Mode Activation Trends**
- Which behavior modes are chosen most often?
- Are there switches mid-task?
- Are certain modes rarely used (e.g., strategic avoidance)?

4. **Persona Interaction**
- Which personas are favored?
- What tone combinations are chosen (e.g., Planner + Motherly)?
- Are certain personas used in emotionally charged moments?

5. **Memory Recall & Prompt Repetition**
- Reuse of past queries
- Topics revisited often
- Questions you ask repeatedly in different forms

---

**III. Profile Generation Logic**

Each user is scored and described on multiple axes:
- Cognitive style (strategic, reactive, avoidant, exploratory)
- Dominant tone (sharp, nurturing, detached, inquisitive)
- Preferred modes (mode usage distribution + time correlation)
- Focus traits (goal orientation, introspection level, chaos vs clarity)
- Latent behavior clusters (identified via GPT pattern recognition)

Example Output:
> “You think in layers but shift tones under pressure. Your assistant use suggests high cognitive control with emotional override tendencies. You favor clarity, but drift toward nurturing support when stressed.”

---

**IV. UX Design**

- **Panel Location:** Settings > Who Am I
- **Visible Elements:**
  - Behavioral graph (mode usage vs tone vs outcome)
  - Persona interaction map
  - GPT-generated personality paragraph
  - Reflection quotes (selected from your own journal)

---

**V. Pro+ Enhancements**

1. **Customize for Me**
- One-click configuration: auto-activates optimal mode + persona stack
- Optionally adjusts journaling prompts and planner layout

2. **Latent Trait Unlocks**
- Shows hidden patterns you haven’t verbalized
- “You ask for clarity but avoid planning. Want to activate Strategic Coach Persona?”

3. **Profile Evolution Timeline**
- View how your psychological profile changes over time
- Compare past “selves” to current behavior

---

**VI. Safeguards & Ethics**
- All data stays local unless user opts in for sync
- Profile is private unless explicitly exported
- Transparency overlays show exactly what data informed each trait
