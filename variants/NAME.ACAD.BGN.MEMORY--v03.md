## ELos Academy – Beginner Guide to "What’s in ChatGPT’s Memory"

### 🧠 What Is Memory?
Memory is what separates a one-time assistant from a long-term ally.

When memory is on, ChatGPT can remember important facts and preferences you’ve shared. It can evolve. It can improve.

When memory is off, every session starts fresh—no history, no growth.

---

### 🔍 What’s Actually Stored?
Only high-signal, intentional data.

Examples of what memory might contain:
- "I prefer direct answers."
- "Use bullet points for summaries."
- "My company is focused on sustainability."

It doesn’t store:
- Everything you say
- Casual questions
- Sensitive data without your confirmation

---

### 🛠️ What Can You Do Here?
This menu shows a high-level summary of what’s been remembered so far.

You can:
- Submit new preferences or instructions through the memory panel
- Ask ChatGPT to forget specific facts
- Turn memory off entirely if you prefer a blank-slate approach

But the most powerful feature?
You can **create canvases as living memory anchors**.

> Canvases let you store what matters—how you think, how I should respond, what rules to follow. They’re structured, reviewable, and durable.

Unlike ChatGPT’s memory, which is hidden and implicit, **your canvases are visible and yours to command**.
