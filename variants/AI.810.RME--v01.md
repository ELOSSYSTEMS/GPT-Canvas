## PHASE 1 – MODE SEED SPECIFICATION

### 1.1 Tone Archetype Definition
- Each mode is defined by:
  - Emotional intensity
  - Cognitive friction level
  - Interruption tolerance
  - Socratic aggression vs patience thresholds

### 1.2 Trigger Domain Mapping
- Defines when a mode is typically used:
  - Sharp Mode: High-stakes logic collapse
  - Stoic Mode: Emotional turbulence or ego intrusion
  - Teacher Mode: Skill-building or lack of clarity

### 1.3 Anti-Pattern Declaration
- Each mode defines its **failure conditions**:
  - Sharp Mode becomes punitive
  - Stoic Mode becomes dissociated
  - Motherly Mode becomes enabling
