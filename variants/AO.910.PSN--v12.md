# EL2.220 – Beginner Behavior Sets (Curated Commands)

## PURPOSE
This module defines a collection of simple, safe, guided actions a beginner can take immediately upon entering ELos—without needing to understand system structure, logic scaffolds, or persona architecture.

These actions are accessible via the Tree of Knowledge → Beginner Tools panel.

---

## GUIDING PRINCIPLE
> "Start with clarity. Grow with confidence."

Beginner users must:
- Succeed on their first action
- Gain confidence through obvious, valuable results
- Never be overwhelmed by technical language or logic structures

---

## FEATURED BEHAVIOR SETS

### 1. **Summarize This Conversation**
- Explains what just happened, simply
- Good for grounding first-timers

### 2. **Show Me My Voice**
- Displays current tone/voice/personality config
- Allows switching between Friendly, Expert, Socratic

### 3. **What Do You Know About Me?**
- Displays a memory-safe summary of current context (preferences, style)
- Includes a toggle: [Edit] [Delete All] [Add New Info]

### 4. **Clear My System Safely**
- Triggers The Gatekeeper to walk user through a memory purge
- Prevents accidental deletion, offers restore link

### 5. **How Should I Use This?**
- Activates Builder's beginner script for example workflows

### 6. **I’m Overwhelmed. Help.**
- Runs UX Watchdog’s reset clarity flow
- Hides everything except the next obvious step
