**System Name:** GPTOS (Working Title)

**Purpose:** A modular Chrome extension that transforms the ChatGPT interface into a personal assistant operating system. GPTOS governs behavior, tone, task planning, journaling, and long-term memory through structured canvases, personas, and modular behavioral overrides.

---

**Core Value Pillars:**
1. **Memory Expansion** – Users simulate persistent assistant memory far beyond ChatGPT's native limits by using structured canvases, journals, and logs. Recall is contextual, organized, and taggable via The Librarian.
2. **Behavior Modes** – Users switch between emotional and strategic interaction styles (Sharp, Motherly, Fatherly, Teacher) to govern the assistant’s tone and response logic in real time.

---

**Key Modules:**
1. **Behavior Modes**
   - Sharp (default)
   - Motherly
   - Fatherly
   - Teacher
   - Only one active at a time; mode is persistent until changed

2. **Persona Management**
   - Activate personas like The Strategist, The Scribe, etc.
   - Create, modify, and deactivate from the sidebar

3. **Module Toggles**
   - Prompt Surgeon
   - Stoic Governor
   - Consistency Auditor
   - Modules run in background unless toggled off

4. **Journaling Tools**
   - The Scribe: Raw thought capture
   - Daily Self-Journaler: Structured planning and reflection
   
5. **Planning & Execution**
   - Daily task breakdown with time blocks
   - Auto-rollover of uncompleted tasks
   - Morning + evening check-ins

6. **Canvas System + The Librarian**
   - Centralized archive of strategic canvases, journal entries, logs
   - Indexed, searchable, and taggable
   - Canvas recall expands assistant memory far beyond native limits

7. **Command Console**
   - Natural language commands interpreted as system instructions
   - Examples: “Activate Fatherly mode,” “What’s left from yesterday?”

---

**Status:** Core structure locked. Canvases, architecture, and extension UI will be developed and versioned under GPTOS.
